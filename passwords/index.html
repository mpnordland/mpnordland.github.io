Secure Password Generator



How does this work? | Made by Maya Nordland: rehack.me

This uses numbers from random.org to generate passwords using the diceware method.

The original diceware method used a set of dice and a long list of short, common words. This still uses the list of words, but instead of dice it uses random numbers from random.org and looks them up automatically for you. The lookup is done completely client side. Obviously, the random numbers must be fetched from random.org. Thus the paranoid should still use dice. But for the rest of us, this should be good.
